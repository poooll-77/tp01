5- ¿Qué es un atributo en una etiqueta HTML? Dar dos ejemplos de etiquetas HTML con atributos, marcando sus respectivos atributos.

Un atributo es una caracteristica que se le asigna a una etiqueta html, ejemplo:

meta charset="UTF-8"

charset: es el atributo. "UTF-8": es la etiqueta.

html lang="es"

lang: es el atributo. "es": es la etiqueta.

6- Investiga qué hacen y cómo se utilizan las siguientes etiquetas HTML: strong, i, u, br, sup, sub, em, span, b, a.

strong: El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto.

i: Muestra el texto marcado con un estilo en cursiva o italica.

u: Muestra el texto subrayado.

br:
Salto de linea.

sup: Define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice.

sub: Define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en ubíndice.

em: Es el apropiado para marcar con énfasis las partes importantes de un texto. El elemento em puede ser anidado, con cada nivel de anidamiento indicando un mayor grado de énfasis.

span: Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea.

b: Pone el texto en negrita.

a: Crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

8- ¿Que es el HTML y en que consiste?

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. Por ejemplo, sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

Las etiquetas HTML son fragmentos de código que permiten crear elementos HTML, estructuras básicas del lenguaje de programación. Funciona tanto en mayúscula como en minúscula

Llamamos elemento a la parte del DOM, modelo de documento para mostrar la página en el browser. Un elemento típico incluye una etiqueta de apertura con algunos atributos, contenido de texto cerrado y una etiqueta de cierre.
